# Exercícios e Jogos

Repositório com exercícios práticos e pequenos jogos desenvolvidos durante meus estudos em ADS (Análise e Desenvolvimento de Sistemas) e cursos complementares.

Cada pasta contém um projeto independente, focado em praticar conceitos específicos como manipulação do DOM, lógica de programação, eventos e estruturação de páginas com HTML/CSS/JavaScript.

Esses projetos foram feitos em momentos diferentes do meu aprendizado, então o nível de complexidade e organização do código varia entre eles — alguns são mais simples, outros mais elaborados. Estão reunidos aqui para manter meu perfil organizado.
